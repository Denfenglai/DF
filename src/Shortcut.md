@tab Ubuntu

```bash
u   # 启动ubuntu 18.04
```

@tab DF

```bash
d   # 启动脚本
```

@tab Yunzai

```bash
yz  # cd根目录
y   # 前台启动
r   # 后台启动
s   # 停止运行
l   # 查看日志
g   # 重置账号
```

@tab Miao-Yunzai

```bash
mz  # cd喵崽根目录
m   # 前台启动喵崽
mr  # 后台启动喵崽
ml  # 查看喵崽日志
ms  # 停止喵崽运行
mg  # 重置喵崽账号
```

::: warning 注意！

所有快捷键均在安装对应功能后才写入
如果没有找到命令多半是写入失败
:::
